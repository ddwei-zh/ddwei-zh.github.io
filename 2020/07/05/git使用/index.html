git的安装与使用

  • https&ssh两种方式
  • [常用的命令](https://www.jianshu.com/p/38921d19ba0a)

– 本地新建一个文件夹，初始化.git后，使用git remote add origin xxx建立仓库联系
– 添加到暂存区git add
– 提交到本地仓库 git commit -m
– 更改提交日志 git –ammend
– 查看本地分支 git branch
– 查看仓库分支 git branch -a
– 使用git pull 更新分支信息
– git checkout -a xx （新建并建立分支）

  • 丢弃修改（本地修改或者暂存区的修改）
  • 储藏与恢复 (用于暂存区与工作区)
  • 回退(reset revert)
